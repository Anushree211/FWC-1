\section*{Question:42}

The logic gate circuit shown below realizes the function \(Z\).

\begin{center}
\includegraphics[width=10cm]{question.jpg}
\end{center}

Possible outputs are:

\begin{itemize}
\item XOR
\item XNOR
\item Half Adder
\item Full Adder
\end{itemize}

\section*{Solution}

From the circuit, NAND gates are used to construct the logic.

The output simplifies to:

\[
Z = X \oplus Y
\]

Thus the circuit implements the \textbf{XOR function}.

\section*{Truth Table}

\hline

\end{tabular}

\section*{Conclusion}

From the analysis of the given logic circuit and its truth table, the output is high only when the inputs \(X\) and \(Y\) are different. This corresponds to the behavior of an XOR gate.

Hence, the correct option is \textbf{(a) XOR}.
