\section*{Question}

What is the Boolean expression for the output $f$ of the combinational logic circuit of NOR gates given below?

\begin{center}
\includegraphics[width=0.8\textwidth]{figs/question.jpg}
\end{center}

\textbf{Options}

\begin{itemize}
\item[a)] $\overline{Q+R}$
\item[b)] $\overline{P+Q}$
\item[c)] $\overline{P+R}$
\item[d)] $\overline{P+Q+R}$
\end{itemize}

\section*{Solution}

Using NOR gate property,

$NOR(A,B) = (A+B)'$

Simplifying the circuit,

$f = \overline{P+Q+R}$

Thus the correct option is \textbf{(d)}.
